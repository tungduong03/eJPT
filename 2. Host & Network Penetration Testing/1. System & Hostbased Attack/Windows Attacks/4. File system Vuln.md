## Alternate Data Streams (ADS)

Là 1 thuộc tính của tệp NTFS (New Technology File System) (NTFS là định dạng ổ đĩa để cài window lên đó)

1 file được tạo trên NTFS format drive sẽ bao gồm:

- Data stream: chứa dữ liệu tệp

- Resource stream: chứa metadata của tệp (thời điểm tệp được tạo, người tạo,...)
